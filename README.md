# DayBook3

Простое приложение на Qt6.

## Как собрать (Windows)

### Если у вас уже есть MSYS2:

1. Откройте **MSYS2 UCRT64**
2. Перейдите в папку проекта
3. Выполните:
   ```bash
   ./build.sh
4. Запустите build/DayBook3.ex

### Если MSYS2 не установлен:

1. Запустите setup.bat двойным кликом
2. Дождитесь окончания установки и сборки
3. Запустите build/DayBook3.exe

### Требования

- Windows 10 или новее
- Интернет (для первоначальной установки MSYS2 и пакетов)
